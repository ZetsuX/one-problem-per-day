# Digits

Source : [Problem](https://www.eolymp.com/en/problems/2)

- Time limit 1 second
- Memory limit 128 MiB

Find the number of digits in a nonnegative integer n.

## Input data

One nonnegative integer n (0 ≤ n ≤ 2 \* 10^9).

## Output data

Print the number of digits in number n.

## Examples

### Sample Input 1

    12343

### Sample Output 1

    5
